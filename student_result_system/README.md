# Student Result System (Python)

This is a simple Python mini project that calculates:

- Student Grades
- SGPA (Semester Grade Point Average)
- CGPA (Cumulative Grade Point Average)

## Features
- Calculates total and average marks
- Assigns grade based on average
- Calculates SGPA using grade points and credits
- Calculates CGPA using semester SGPA values

## Technologies Used
- Python
- Basic programming concepts
  - Functions
  - Conditional statements
  - Loops
  - User input

## How to Run the Program
1. Download the project
2. Open the file `student_results_system.py`
3. Run the program using Python

Example:

```bash
python student_results_system.py
